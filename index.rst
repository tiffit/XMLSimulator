RotMG XML Simulator Documentation
=================================

The RotMG XML Simulator allows you to simulate RotMG Enemy XML. The sim also comes with many features to help with development and debugging.
The purpose of this documentation is to help you learn how to use the Simulator and all of its features.

**Do not share the XML simulator itself outside of the testing group.**
